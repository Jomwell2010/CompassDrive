# 🚗 CompassDrive

An AI-powered driving tutor that helps users prepare for the California driving test with personalized, reliable guidance.

🔗 **Live Demo:** https://compass-drive.replit.app

---

## Features

* 🤖 AI tutor that answers California driving questions
* 📚 RAG-powered knowledge retrieval for more accurate responses
* 🚦 Explains road signs, traffic laws, and safe driving practices in simple language
* 🎯 Personalized learning to help users study with confidence
* 💬 Natural conversations with follow-up questions for deeper understanding

---

## How It Works

When a user asks a question, CompassDrive searches a curated driving knowledge base using embeddings and vector search to find the most relevant information. That context is then provided to a large language model (LLM), allowing it to generate accurate, grounded answers while reducing hallucinations.
